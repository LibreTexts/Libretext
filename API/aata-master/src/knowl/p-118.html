Suppose that

\begin{align*} A & = \{ x : x \in \mathbb N \text{ and } x \text{ is even} \},\\ B & = \{x : x \in \mathbb N \text{ and } x \text{ is prime}\},\\ C & = \{ x : x \in \mathbb N \text{ and } x \text{ is a multiple of } 5\}\text{.} \end{align*}

Describe each of the following sets.

  1. \(A \cap B\)

  2. \(B \cap C\)

  3. \(A \cup B\)

  4. \(A \cap (B \cup C)\)
