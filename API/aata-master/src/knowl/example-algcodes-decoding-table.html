Example 8.4.10.

Table 8.4.8 is a decoding table for the code \(C\) given in Example 8.4.5. If \({\mathbf x} = (01111)\) is received, then its syndrome can be computed to be

\begin{equation*} H {\mathbf x} = \begin{pmatrix} 0 \\ 1 \\ 1 \end{pmatrix}\text{.} \end{equation*}

Examining the decoding table, we determine that the coset leader is \((00010)\text{.}\) It is now easy to decode the received codeword.
