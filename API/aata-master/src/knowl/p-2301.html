The proofs of the Second and Third Sylow Theorems (Theorem 15.1.7, Theorem 15.1.8) employ a group action on sets of Sylow \(p\)-subgroups, For the Second Theorem, the list is proposed as incomplete and is proved to be all of the Sylow \(p\)-subgroups. In this exercise we will see how these actions behave, and how they are different when we use different groups acting on the same set.
