What have we accomplished? In the first line we create pas the product of two group elements, one from coset \(1\) and one from coset \(3\) (C[1], C[3]). Since we can choose any element from each coset, we choose the first element of each (C[ ][0]). Then we count our way through all the cosets, selecting only cosets that contain p. Since pwill only be in one coset, we expect a list with just one element. Here, our one-element list contains only 2. So we say the product of coset \(1\) and coset \(3\) is coset \(2\text{.}\)
