Theorem 8.2.3.

Let \(d_{\min}\) be the minimum distance for a group code \(C\text{.}\) Then \(d_{\min}\) is the minimum of all the nonzero weights of the nonzero codewords in \(C\text{.}\) That is,

\begin{equation*} d_{\min} = \min\{ w({\mathbf x}) : { {\mathbf x} \neq {\mathbf 0} } \}\text{.} \end{equation*}
Proof.

Observe that

\begin{align*} d_{\min} & = \min \{ d({\mathbf x},{\mathbf y}) : {\mathbf x}\neq{\mathbf y} \}\\ &= \min \{ d({\mathbf x},{\mathbf y}) : {\mathbf x}+{\mathbf y} \neq {\mathbf 0} \}\\ &= \min\{ w({\mathbf x} + {\mathbf y}) : {\mathbf x}+{\mathbf y}\neq {\mathbf 0} \}\\ & = \min\{ w({\mathbf z}) : {\mathbf z} \neq {\mathbf 0} \}\text{.} \end{align*}
