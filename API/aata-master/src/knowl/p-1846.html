\((4) \Rightarrow (5)\text{.}\) If \(A\) is distance-preserving, then \(A\) is length-preserving. Letting \({\mathbf y} = 0\text{,}\) we have

\begin{equation*} \| A{\mathbf x}\| = \| A{\mathbf x}- A{\mathbf y} \| = \| {\mathbf x}- {\mathbf y} \| = \| {\mathbf x} \|\text{.} \end{equation*}
