Any element fixed by tauwill be a linear combination of these four elements. We can ignore any rational multiples present, the first element is just saying the rationals are fixed, and the last element is just a product of the middle two. So fundamentally tauis fixing rationals, b (which is \(\sqrt[4]{2}i\)) and a^2 (which is \(\sqrt{2}\)). Furthermore, b^2 = -a^2 (the check follows), so we can create any fixed element of tauby just adjoining b=\(\sqrt[4]{2}i\) to the rationals. So the elements fixed by tauare \({\mathbb Q}(\sqrt[4]{2}i)\text{.}\)
