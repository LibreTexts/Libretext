To prove this theorem, we will employ the method that we used to construct Example 21.1.2. Clearly, we can assume that \(p(x)\) is an irreducible polynomial. We wish to find an extension field \(E\) of \(F\) containing an element \(\alpha\) such that \(p(\alpha) = 0\text{.}\) The ideal \(\langle p(x) \rangle\) generated by \(p(x)\) is a maximal ideal in \(F[x]\) by Theorem 17.3.12; hence, \(F[x]/\langle p(x) \rangle\) is a field. We claim that \(E = F[x]/\langle p(x) \rangle\) is the desired field.
