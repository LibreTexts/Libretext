Proof

We know that \(|G|/|G_x|\) is the number of left cosets of \(G_x\) in \(G\) by Lagrange's Theorem (TheoremÂ 6.2.2). We will define a bijective map \(\phi\) between the orbit \({\mathcal O}_x\) of \(X\) and the set of left cosets \({\mathcal L}_{G_x}\) of \(G_x\) in \(G\text{.}\) Let \(y \in {\mathcal O}_x\text{.}\) Then there exists a \(g\) in \(G\) such that \(g x = y\text{.}\) Define \(\phi\) by \(\phi( y ) = g G_x\text{.}\) To show that \(\phi\) is one-to-one, assume that \(\phi(y_1) = \phi(y_2)\text{.}\) Then

\begin{equation*} \phi(y_1) = g_1 G_x = g_2 G_x = \phi(y_2)\text{,} \end{equation*}

where \(g_1 x = y_1\) and \(g_2 x = y_2\text{.}\) Since \(g_1 G_x = g_2 G_x\text{,}\) there exists a \(g \in G_x\) such that \(g_2 = g_1 g\text{,}\)

\begin{equation*} y_2 = g_2 x = g_1 g x = g_1 x = y_1; \end{equation*}

consequently, the map \(\phi\) is one-to-one. Finally, we must show that the map \(\phi\) is onto. Let \(g G_x\) be a left coset. If \(g x = y\text{,}\) then \(\phi(y) = g G_x\text{.}\)
