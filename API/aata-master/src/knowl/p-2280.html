We saw earlier Sage's permutation group method .is_simple(). Example 15.2.8 tells us that a group of order \(64\) is never simple. The dicyclic group DiCyclicGroup(16)is a non-abelian group of \(64\text{,}\) so we can test this method on this group. It turns out this group has many normal subgroups — the list will always contain the trivial subgroup and the group itself, so any number exceeding \(2\) indicates a non-trivial normal subgroup.
