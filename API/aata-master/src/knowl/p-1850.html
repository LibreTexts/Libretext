An isometry or rigid motion in \({\mathbb R}^n\) is a distance-preserving function \(f\) from \({\mathbb R}^n\) to \({\mathbb R}^n\text{.}\) This means that \(f\) must satisfy

\begin{equation*} \| f({\mathbf x}) - f({\mathbf y}) \| =\|{\mathbf x} - {\mathbf y} \| \end{equation*}

for all \({\mathbf x}, {\mathbf y} \in {\mathbb R}^n\text{.}\) It is not difficult to show that \(f\) must be a one-to-one map. By Theorem 12.1.8, any element in \(O(n)\) is an isometry on \({\mathbb R}^n\text{;}\) however, \(O(n)\) does not include all possible isometries on \({\mathbb R}^n\text{.}\) Translation by a vector \({\mathbf x}\text{,}\) \(T_{\mathbf y}({\mathbf x}) = {\mathbf x} + {\mathbf y}\) is also an isometry (Figure 12.1.11); however, \(T\) cannot be in \(O(n)\) since it is not a linear map.
