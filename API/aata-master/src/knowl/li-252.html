Item 5.5.4.e.

Number the six faces of the cube with the numbers \(1\) through \(6\) (any way you like). Now consider the same three symmetries we used before (quarter-turns about face-to-face axes), but now view them as permutations of the six faces. In this way, we construct each symmetry as an element of \(S_6\text{.}\) Verify that the subgroup generated by these symmetries is the whole symmetry group of the cube. Again, rather than using three generators, try using just two.
