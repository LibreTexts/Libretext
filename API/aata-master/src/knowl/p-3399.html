The number field Ahas been constructed mathematically as what we would write as \({\mathbb Q}\subset{\mathbb Q}[\sqrt{3}]\subset{\mathbb Q}[\sqrt{3},\sqrt{2}]\text{.}\) Notice the slight difference in ordering of the elements we are adjoining, and notice how the number fields use slightly fancier internal names (sqrt2, sqrt3) for the new elements.
