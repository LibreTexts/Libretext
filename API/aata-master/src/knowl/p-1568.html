So we have a few more examples to work with, let us introduce the Sage command that creates an external direct product. If Gand Hare two permutation groups, then the command direct_product_permgroups([G,H])will return the external direct product as a new permutation group. Notice that this is a function (not a method) and the input is a list. Rather than just combining two groups in the list, any number of groups can be supplied. We illustrate isomorphism testing and direct products in the context of Theorem 9.2.9, which is an equivalence, so tells us exactly when we have isomorphic groups. We use cyclic permutation groups as stand-ins for \({\mathbb Z}_n\) by Theorem 9.1.8.
