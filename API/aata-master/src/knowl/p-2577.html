Recall that the division algorithm for integers (Theorem 2.2.1) says that if \(a\) and \(b\) are integers with \(b \gt 0\text{,}\) then there exist unique integers \(q\) and \(r\) such that \(a = bq + r\text{,}\) where \(0 \leq r \lt b\text{.}\) The algorithm by which \(q\) and \(r\) are found is just long division. A similar theorem exists for polynomials. The division algorithm for polynomials has several important consequences. Since its proof is very similar to the corresponding proof for integers, it is worthwhile to review Theorem 2.2.1 at this point.
